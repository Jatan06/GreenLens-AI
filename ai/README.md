# GreenLens AI

## Features

- Waste Detection using YOLOv8
- Recommendation Engine
- Carbon Impact Calculation
- Reward Points
- Bounding Box Detection
- Annotated Image Generation
- FastAPI REST API

## Run

```bash
pip install -r requirements.txt
uvicorn api.main:app --reload
```
